# Sky Fitness Pro

Онлайн-платформа фитнес-курсов: каталог курсов, запись на курсы, личный кабинет с прогрессом и тренировками.

## О проекте

Пользователь может просматривать курсы на главной, открывать детальную страницу курса (описание, направления, запись), входить в аккаунт и добавлять курсы в «Мои курсы». В профиле отображаются выбранные курсы, прогресс по каждому курсу и кнопки «Начать тренировки» / «Продолжить». Выбор конкретной тренировки и ввод прогресса по упражнениям реализованы через модальные окна. Есть адаптивная вёрстка под десктоп, планшет и мобильные устройства.

**Основные страницы:** главная (каталог), детали курса, профиль пользователя, страница тренировки (видео + упражнения с прогрессом).

## Стек

- **Next.js 16** (App Router), **React 19**, **TypeScript**
- **Стили:** CSS Modules, общие стили в `globals.css`
- **Шрифт:** Roboto (next/font)
- **HTTP:** Axios
- **Уведомления:** react-toastify
- **Скелетоны загрузки:** react-loading-skeleton
- **Тесты:** Jest, React Testing Library
- **Линтинг и формат:** ESLint, Prettier
